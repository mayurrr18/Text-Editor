# 📝 Java Text Editor

A simple and lightweight **Text Editor application** built using Java Swing. This project allows users to open, edit, and save text files through an easy-to-use graphical interface.

---

## 🚀 Features

* 📂 Open `.txt` files from your system
* 💾 Save text content to a file
* 🖊️ Edit text in a clean interface
* 📜 Scrollable text area for large content
* 🎯 Simple and beginner-friendly design

---

## 🛠️ Technologies Used

* **Java**
* **Swing (GUI Framework)**
* **AWT (Event Handling)**
* **File Handling (FileReader, FileWriter, BufferedReader)**

## ▶️ How to Run

### Step 1: Clone the repository

```bash
git clone https://github.com/your-username/java-text-editor.git
```

### Step 2: Navigate to the folder

```bash
cd java-text-editor
```

### Step 3: Compile the program

```bash
javac Text_Editor.java
```

### Step 4: Run the program

```bash
java Text_Editor
```
